The [GraphQL Code Generator project](https://www.graphql-code-generator.com/) supports generating clients for `Node.js`, `C#` and `Java`.

## Schema

The GraphQL schema for codegen can be fetched by making a `GET` request to the GraphQL Endpoint `https://signatures-api.criipto.com/v1/graphql`

```text
curl https://signatures-api.criipto.com/v1/graphql
```

## Node.js

This example uses [GraphQL Code Generator](https://www.graphql-code-generator.com/) and is available at [GitHub](https://github.com/criipto/samples/tree/master/Signing/nodejs)

## .NET (dotnet)

A [C#/.NET SDK](https://github.com/criipto/criipto-signatures-dotnet) is provided and maintained by Criipto.
Follow along in [tutorial](/signatures/integrations/csharp/) or see all our [C#/.NET examples](/signatures/graphql/examples/).
