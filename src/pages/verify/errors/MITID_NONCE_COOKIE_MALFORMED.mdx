title: MITID_NONCE_COOKIE_MALFORMED
layout: verify-error
---

MitID has a requirement that the login session must be bound to the user device.
Criipto Verify uses a cookie for this requirement

The `MITID_NONCE_COOKIE_MALFORMED` error code indicates that the required cookie
is present, but not in the shape we expect it to be - indicating a potential malicious cookie.
